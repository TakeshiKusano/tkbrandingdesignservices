# コミュニケーション展開ページ（コミュニケーション展開.html）用の画像フォルダ

このフォルダに画像ファイル（jpg または png）を保存し、`コミュニケーション展開.html` 内の該当する
`<img src="communication/ファイル名">` の src をそのファイル名に合わせるだけで表示されます。
画像がない項目は、グラデーション背景のプレースホルダーとして表示されます。

## 現在使用中のファイル名

| ファイル名 | 使用箇所 |
|:---|:---|
| `entry-01.jpg` | エントリー1: グリーンビーンズ ブランド訴求TVCM |
| `entry-02.jpg` | エントリー2: 大規模PRイベントの企画・実施（4月4日記者発表・7月10日CFCグランドオープンの2スライドを1枚に結合） |
| `entry-03.jpg` | エントリー3: 2024年 PRファクトブック・事業案内小冊子（全8ページを1枚に結合） |
| `entry-04.jpg` | エントリー: 2023年5月 イオン株主総会支援（展示ブース写真5点を1枚に結合） |

推奨サイズ: 横1200×縦514px程度（横長21:9）

## 実績を追加したい場合

`コミュニケーション展開.html` の `<!-- Entry Section -->` 内にある
`<div class="glass-card">〜</div>` のブロックをまるごとコピーし、
画像ファイル名・見出し（h3）・文章（p）を書き換えて追加してください。
