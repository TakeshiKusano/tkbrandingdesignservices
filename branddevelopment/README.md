# ブランド開発ページ（Branddevelopment.html）用の画像フォルダ

このフォルダに画像ファイル（jpg または png）を保存し、`Branddevelopment.html` 内の該当する
`<img src="branddevelopment/ファイル名">` の src をそのファイル名に合わせるだけで表示されます。
画像がない項目は、グラデーション背景のプレースホルダーとして表示されます。

## 現在使用中のファイル名

| ファイル名 | 使用箇所 |
|:---|:---|
| `entry-01.jpg` | エントリー1: グリーンビーンズ ブランドアイデンティティ開発（GreenBeansブランド俯瞰図） |

推奨サイズ: 横1200×縦514px程度（横長21:9）

## 実績を追加したい場合

`Branddevelopment.html` の `<!-- Entry Section -->` 内にある
`<div class="glass-card">〜</div>` のブロックをまるごとコピーし、
画像ファイル名・見出し（h3）・文章（p）を書き換えて追加してください。
