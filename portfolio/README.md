# 実績ページ（portfolio.html）用の画像フォルダ

このフォルダに指定のファイル名で画像を保存するだけで、`portfolio.html` に自動的に表示されます。
コードの編集は不要です。画像がない項目は、金色のグラデーション背景がプレースホルダーとして表示されます。

## 事例カードの画像（推奨サイズ: 横1200×縦514px程度、横長21:9）

| ファイル名 | 使用箇所 |
|:---|:---|
| `case-greenbeans.jpg` | 事例1: イオンネクスト「グリーンビーンズ」 |
| `case-topvalu.jpg` | 事例2: イオントップバリュ |
| `case-independent.jpg` | 事例3: 個人事業主としての伴走支援 |

## ギャラリーの画像（推奨サイズ: 横1200×縦900px程度、4:3）

| ファイル名 | 使用箇所 |
|:---|:---|
| `gallery-01.jpg` | ギャラリー1枚目 |
| `gallery-02.jpg` | ギャラリー2枚目 |
| `gallery-03.jpg` | ギャラリー3枚目 |

## 画像・カードを追加したい場合

`portfolio.html` 内の該当セクションに使い方のコメントを書いてあります。
- 事例カードを増やす: `<!-- Case Studies Section -->` 内の `<div class="glass-card">〜</div>` をコピーして書き換え
- ギャラリー項目を増やす: `<!-- Gallery Section -->` 内の `<div class="gallery-item">〜</div>` をコピーして書き換え

## 文章を追記したい場合

各事例カードの末尾に、コメントアウトされた文章追加用の `<p>` タグを用意してあります。
コメント（`<!-- -->`）を外して自由に書き足してください。
